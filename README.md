# Python Security Labs

A structured collection of security-oriented Python exercises and controlled lab scripts.

This repository is focused on strengthening practical scripting skills relevant to cybersecurity and system analysis.

## Focus Areas

- Log parsing and pattern detection  
- Input validation logic  
- File handling and structured data parsing  
- Small security-oriented utility scripts  

## Repository Structure

- `log-analysis/` – Exercises involving log inspection and pattern counting  
- `input-validation/` – Safe handling of user-provided data  
- `file-parsing/` – Working with structured and semi-structured files  
- `mini-tools/` – Standalone utility scripts  

This repository represents ongoing refinement of secure scripting fundamentals.
